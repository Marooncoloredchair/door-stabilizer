# Examples

Run from a virtualenv with `door-stabilizer` installed:

```bash
pip install door-stabilizer
python minimal_door_loop.py
```

Learning-curve plot (Door vs random on a toy plant; needs matplotlib):

```bash
pip install door-stabilizer matplotlib
python plot_learning_curve.py
```

For optional PyTorch:

```bash
pip install "door-stabilizer[torch]"
```
