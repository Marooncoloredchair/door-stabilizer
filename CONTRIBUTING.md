# Contributing

- **Bug reports**: Open an issue with Python version, `door-stabilizer` version (`pip show door-stabilizer`), and a minimal reproducer.
- **Docs**: Pull requests that clarify install steps or examples are welcome.
- **Upstream**: Core library development may live in a larger research tree; this repo tracks **user-facing** documentation and examples aligned with [PyPI — door-stabilizer](https://pypi.org/project/door-stabilizer/).
